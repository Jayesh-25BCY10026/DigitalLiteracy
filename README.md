Project Overview->
This project is based on digital literacy and its importance for students in today’s time. As a student digital ambassador, the main goal was to learn and apply different digital tools, improve online communication and understand cyber safety.
The project is divided into 5 tasks, each focusing on a different aspect like infographic creation, digital profiles, platforms, email etiquette and cybercrime awareness.

Task 1 – 
Created a digital literacy infographic using Canva covering important topics like tools, safety and basics. It was simple but useful for understanding.

Task 2 – 
Created and updated profiles on GitHub and LinkedIn. These platforms are important for showcasing skills and future opportunities.

Task 3 – 
Solved a beginner problem on HackerRank and created a Google Form quiz. It helped in learning practical usage of tools.
Google Form Link: https://docs.google.com/forms/d/e/1FAIpQLSc2rV8fw81VAHb7vbcTus9zzk04RDBM_eloLl9LNPvqqdvukQ/viewform?usp=publish-editor

Task 4 – 
Wrote two professional emails and created a checklist of social media do’s and don’ts. Learned how proper communication is important.

Task 5 – Cybercrime Awareness
Worked on phishing case study and created a prevention checklist. It showed how easily people can get scammed if not careful.
